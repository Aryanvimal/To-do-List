# Java To‑Do List (CLI)

A simple **command‑line To‑Do List application** built in Java.  
You can add tasks, view them, mark them as done, and delete them.

## How to Run

```bash
javac src/*.java
java -cp src Main
```

## Features
✔ Add Tasks  
✔ View Tasks  
✔ Mark Tasks as Done  
✔ Delete Tasks  
✔ Runs in any terminal
